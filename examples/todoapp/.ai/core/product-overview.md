# Product Overview

## Product Name
TodoApp

## Purpose
A simple, single-page todo list web application that allows users to manage their tasks efficiently.

## Target User
Anyone needing a lightweight, no-frills task management tool in their browser.

## Core Action
Manage tasks — add new tasks, mark them as complete, and delete them.

## Differentiator
Zero dependencies, pure vanilla HTML/CSS/JS — no frameworks, no build tools, instant load.

## Screens
- **Single Page**: One unified view containing:
  - Input field for adding new tasks
  - Task list displaying all tasks with completion status
  - Controls for completing and deleting individual tasks

## Acceptance Criteria
- User can add a new task via text input
- User can mark a task as complete (visual distinction for completed tasks)
- User can delete a task
- All interactions happen on a single page without navigation
- No external frameworks or libraries — vanilla HTML, CSS, and JavaScript only
