# Glossary

| Term | Definition |
|------|-----------|
| Task | A single to-do item created by the user, containing text and a completion state |
| Add | Creating a new task and appending it to the task list |
| Complete | Marking a task as done, visually distinguishing it from pending tasks |
| Delete | Permanently removing a task from the list |
| Task List | The ordered collection of all tasks displayed on the page |
| SPA | Single Page Application — the entire app runs on one HTML page with no navigation |
